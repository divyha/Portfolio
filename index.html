Description:
Applicant Tracking System is the desktop application developed through Netbeans in java programming Language.
It is a CRUD application for a staffing company for managing their database in an appropriate manner and gene-
rating a report which is achieved by Jasper iReport.This Application enables the electronic handling of recru-
itment needs. It is an application for an enterprise or small business level, depending on the needs of the
company. It is basically designed for maintaining the records of the sourced candidates. In many cases we can
filter applications automatically based on given criteria such as keywords, Position,Clients and Location
